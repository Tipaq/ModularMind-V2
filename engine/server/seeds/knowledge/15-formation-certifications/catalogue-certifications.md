## 1. Politique de Certification

ModularMind encourage activement ses collaborateurs à obtenir des certifications professionnelles reconnues. Les certifications figurant dans ce catalogue bénéficient d'une **prise en charge à 100 % des frais d'examen** (hors budget formation individuel de 2 000 euros), d'un **jour de préparation** (congé formation la veille de l'examen), et d'une **prime de réussite de 200 euros** versée sur le bulletin de paie du mois suivant.

En cas d'échec au premier passage, un second passage est pris en charge à 50 %. Au-delà, les frais sont à la charge du collaborateur ou imputés sur son budget formation individuel.

Les certifications obtenues sont :
- Affichées sur le profil interne (Notion > Équipes > Profils)
- Annoncées sur le canal Slack `#certifications`
- Prises en compte dans les critères de promotion (voir organigramme-roles.md)

## 8. Parcours Recommandés par Rôle

### Backend Engineer (Python)

```
Année 1 : PCAP + AWS Cloud Practitioner + Terraform Associate
Année 2 : AWS Solutions Architect Associate + CKA + CompTIA Security+
Année 3 : AWS ML Specialty ou GCP ML Engineer + PCPP1
```

### Frontend Engineer (TypeScript/React)

```
Année 1 : AWS Cloud Practitioner + GitHub Foundations
Année 2 : GitHub Actions + GCP Cloud Digital Leader
Année 3 : Azure Fundamentals + spécialisation au choix
```

### DevOps / SRE

```
Année 1 : CKA + Terraform Associate + AWS Solutions Architect Associate
Année 2 : CKAD + Vault Associate + GitHub Actions
Année 3 : CKS + AWS DevOps Professional + CISSP (si orientation sécurité)
```

### ML Engineer

```
Année 1 : AWS ML Specialty + PCAP + TensorFlow Developer

```

```
Année 1 : CompTIA Security+ + CKS + Vault Associate
```

## 9. Procédure de Demande

1. **Choisir une certification** dans ce catalogue (ou soumettre une demande pour une certification hors catalogue)
2. **En discuter avec son manager** lors d'un 1:1 pour valider l'alignement avec le PDI
3. **S'inscrire à l'examen** et soumettre la facture via Payfit (catégorie "Certification")
4. **Préparer l'examen** en utilisant les ressources listées (licences entreprise disponibles)
5. **Poser le jour de préparation** la veille de l'examen via Payfit (type "Congé formation")
6. **Passer l'examen** et partager le résultat (succès ou échec) avec le manager et les RH
7. **En cas de réussite** : communiquer sur `#certifications`, prime de 200 euros versée automatiquement
