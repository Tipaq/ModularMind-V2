# Guide d'onboarding — Nouveaux employés ModularMind

## Bienvenue chez ModularMind !

Félicitations pour votre arrivée dans l'équipe ! Ce guide vous accompagne dans vos premiers jours et semaines pour une intégration réussie.

## Jour 1 — Accueil

### Matin
- **9h00** : Accueil par votre manager et visite des locaux
- **9h30** : Remise du matériel (laptop, badge, écran)
- **10h00** : Session RH (contrat, mutuelle, tickets restaurant)
- **10h30** : Création des comptes (email, Slack, GitHub, Jira)
- **11h00** : Présentation de votre buddy (mentor d'intégration)

### Après-midi
- **14h00** : Présentation de l'entreprise par le CEO (30 min, visio)
- **14h30** : Installation de l'environnement de travail avec votre buddy
- **16h00** : Tour des équipes (présentations informelles)
- **17h00** : Débrief avec votre manager

## Plan 30-60-90 jours

### 30 premiers jours — Apprentissage
- Comprendre le produit ModularMind et ses utilisateurs
- Maîtriser l'environnement de développement
- Livrer 2-3 petites tâches (bug fixes, docs)
- Participer aux cérémonies agile (daily, sprint review)
- Rencontrer individuellement chaque membre de l'équipe

### 60 premiers jours — Contribution
- Prendre en charge des features de taille moyenne
- Participer aux code reviews (reviewer et auteur)
- Comprendre le pipeline CI/CD et le processus de déploiement
- Contribuer à la documentation technique
- Identifier des améliorations potentielles

### 90 premiers jours — Autonomie
- Mener un projet de bout en bout
- Être autonome sur le workflow de développement complet
- Participer aux discussions d'architecture
- Mentorer (optionnel) un nouvel arrivant si applicable
- Rétrospective 90 jours avec le manager
